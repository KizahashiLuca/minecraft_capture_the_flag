## ルール

### ゲームの流れ
 1. ホストプレイヤーが、以下コマンドを実行
    ```mcfunction
    function #mcf:start
    ```
 2. 制限時間、ワールドボーダーを設定
 3. ゲーム開始時、参加者がランダムな場所にテレポート
 4. 全参加者が着地した時点から、制限時間スタート
